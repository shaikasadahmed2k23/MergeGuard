How It Works
01 — WEBHOOK
PR Event Received
GitHub fires a signed webhook. MergeGuard verifies the HMAC signature and fetches the full diff.
02 — CLASSIFY
Blast Radius Check
A deterministic classifier scans for risk signals before any LLM call. High-risk areas are never auto-merged, regardless of score.
03 — ANALYZE
5 Agents in Parallel
Security, Intent, Diff, Impact, and Context agents run simultaneously via Google ADK + Gemini 2.5 Flash.
04 — DECIDE
Autonomous Decision
Weighted trust score is calculated. Agent merges, rejects, or escalates — and explains every factor in the PR comment.
